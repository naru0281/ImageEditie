# ImageEditie

ImageEditie is a lightweight browser-based image editing tool built with **HTML, CSS, and JavaScript**.
It allows users to quickly resize images or adjust file size directly in the browser without requiring any server-side processing.

All processing is performed **locally in the user's browser**, ensuring privacy and fast performance.

---

# Features

## Image Size Adjustment

* Drag & Drop image upload
* Supported formats: **JPG / JPEG / PNG / WEBP / GIF / BMP**
* Image preview
* Custom output size input (px)
* Adjustable cropping area
* Download cropped image

## File Size Adjustment

* Drag & Drop image upload
* Target file size input (byte)
* Quick conversion buttons for **KB / MB**
* Automatic compression using browser canvas encoding
* Download compressed image

---

# UI

* Discord-style interface
* Light mode / Dark mode toggle
* iOS-style segmented control with slide animation
* Toast notifications for errors and completion

---

# Technology

* HTML
* CSS
* JavaScript
* Canvas API

No external libraries or frameworks are used.

---

# Usage

1. Open `index.html` in a browser.
2. Select the editing mode using the segmented control:

   * **Image Size Adjustment**
   * **File Size Adjustment**
3. Drag & drop an image or select a file.
4. Enter the desired size or file size.
5. Download the processed image.
